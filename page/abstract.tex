\addcontentsline{toc}{chapter}{Abstract} % Add Abstract to Table of Contents
Analyze the malicious payload in the different samples of APKs.
The report must include the following sections:
\begin{itemize}
    \item Anti malware analysis: by submitting the samples to the tool web VirusTotal and discussing the result.
\item Static analysis: by using Bytecodeviewer or JD-GUI to highlight interesting snippet of code (include and discuss such code in the report) and by using MobSF framework to extract automatically features from the code interesting for MobSF malware analysis.
\item (Optional) Dynamic analysis: execute the application in a controlled environment (a simulator) and observe the behavior of the application.
\item The APKs Will be provided on request (There is a small amount of available APK...)
\end{itemize}

This project does not involve the submission of the code.
